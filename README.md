# architecture-nodejs-base

## Tuto base NodeJS

## Pré-requis

NodeJS v10
MongoDB Local ou Cloud (db: apiRest)

## Installer l'appli

```
sudo mongod
```
## Nouveau Terminal

```
git clone https://github.com/xdrkush/architecture-nodejs-base.git
cd architecture-nodejs-base
npm i
npm start
```
