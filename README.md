# architecture-nodejs-base

En Cours !!

## Tuto base NodeJS

# Pré-requis

NodeJS v10
MongoDB Local ou Cloud (db: apiRest) Nodemon

# Installer Nodemon
```
sudo npm i nodemon -g
```

# Installer l'appli

```
sudo mongod
```

## Nouveau Terminal

```
git clone https://github.com/xdrkush/architecture-nodejs-base.git
cd architecture-nodejs-base
npm i
npm start
```
